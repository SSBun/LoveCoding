# Web 开发

欢迎来到 Web 开发文档中心。本章节将涵盖主流前端框架 React、Vue 以及 TypeScript 的使用。

## 目录

- [React](./react) - Facebook 出品的 UI 库
- [Vue](./vue) - 渐进式 JavaScript 框架
- [TypeScript](./typescript) - JavaScript 的超集

## Web 开发技术栈

现代 Web 开发通常涉及以下技术：

- **HTML/CSS** - 页面结构和样式
- **JavaScript/TypeScript** - 交互逻辑
- **React/Vue/Angular** - 前端框架
- **Node.js** - 服务端 JavaScript
- **Webpack/Vite** - 构建工具

## 学习路线

1. 掌握 HTML 和 CSS 基础
2. 学习 JavaScript 核心概念
3. 了解 TypeScript
4. 选择一个前端框架深入学习
5. 掌握构建工具和工程化
