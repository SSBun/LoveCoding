# Server 端开发

欢迎来到服务端开发文档中心。本章节将涵盖 Node.js、Python 以及数据库相关技术。

## 目录

- [Node.js](./nodejs) - JavaScript 运行时
- [Python](./python) - 多用途编程语言
- [数据库](./database) - 数据存储与管理

## 服务端技术概览

服务端开发涉及以下核心领域：

- **运行时环境** - Node.js, Python, Go, Java 等
- **数据库** - MySQL, PostgreSQL, MongoDB, Redis
- **API 设计** - REST, GraphQL, gRPC
- **容器化** - Docker, Kubernetes
- **云服务** - AWS, GCP, Azure

## 学习路线

1. 掌握一门服务端语言（Node.js 或 Python）
2. 学习数据库基础
3. 理解 API 设计和安全
4. 掌握容器化和部署
5. 了解微服务架构
