# Android 开发

欢迎来到 Android 开发文档中心。本章节将涵盖 Kotlin 编程语言和 Jetpack 框架的使用。

## 目录

- [Kotlin](./kotlin) - Android 开发首选语言
- [Jetpack](./jetpack) - Android 开发组件库

## 什么是 Android？

Android 是谷歌公司开发的移动操作系统，基于 Linux 内核。使用 Kotlin 可以构建原生 Android 应用程序。

## 环境要求

- JDK 17+
- Android Studio Arctic Fox+
- Gradle 8.0+

## 创建项目

使用 Android Studio 创建新项目，选择 "Empty Views Activity" 模板。
